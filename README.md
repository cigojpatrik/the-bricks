# The Bricks

**The Bricks** je preprosta arkadna igra narejena z **HTML**, **CSS**, **JavaScript** in **jQuery**.  
Igralec upravlja paddle, odbija žogico in uničuje bricke skozi več levelov.

## Kaj igra vsebuje

- premikanje paddle-a z levo in desno puščico
- začetek igre s **Start** gumbom 
- **Reset** gumb za ponovni začetek igre
- **Pause** gumb za pavzo in nadaljevanje igre
- več levelov z različnim številom brickov
- sistem točk
- časovnik
- SweetAlert ob koncu levela
- space / galaxy vizualna tema
- ozadje in slike za boljši izgled igre

## Kontrole

- **Leva puščica** – premik levo
- **Desna puščica** – premik desno
- **Start** – začetek levela
- **Pause** – pavza / nadaljevanje
- **Reset** – začetek igre od začetka

## Tehnologije

- CSS
- JavaScript
- jQuery
